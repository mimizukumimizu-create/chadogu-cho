# 茶道具帖 — 茶道具の管理・取り合わせアプリ

**アプリのURL 👉 https://mimizukumimizu-create.github.io/chadogu-cho/**

無料・会員登録不要。スマホでもパソコンでも、ブラウザで開くだけで使えます。

## できること

- 茶道具の登録（釜・茶碗・棗・香合・茶杓・柄杓・花入・皿・その他）— 写真も複数枚OK
- 購入時期・購入場所・金額、お手入れの履歴をメモ
- タグ（例：楽焼、初釜）と季節（炉・風炉）で絞り込み
- 道具の「取り合わせ」を組んで、名前を付けて保存

## スマホでの最初の設定（おすすめ）

1. Safari（iPhoneの場合）で上のURLを開く
2. 画面下の共有ボタン（□に↑）→「**ホーム画面に追加**」

ホーム画面のアイコンから、アプリのように起動できるようになります。
※後述しますが、ホーム画面から使うほうがデータも消えにくく安心です。

## ⚠️ いちばん大事な注意：データは「その端末の中」にあります

登録した道具や写真は、**お使いの端末のブラウザの中**に保存されます。
どこかのサーバーに送られることはないので安心な反面、次の点に注意してください。

- **ブラウザの「履歴・サイトデータを消去」をすると、道具のデータも一緒に消えます**
- スマホとパソコンでデータは別々です（自動では同期されません）
- プライベートブラウズ（シークレットモード）で開くと保存されません。必ず通常モードで
- iPhoneのSafariで「ブックマークから開くだけ」の使い方だと、1週間以上使わない場合にデータが自動削除されることがあります。**ホーム画面に追加して使えばこの心配はありません**

## 定期バックアップのやり方（月1回くらいが安心）

1. 「道具一覧」タブのいちばん下にある「**データを書き出す（バックアップ）**」を押す
2. 「茶道具帖バックアップ_日付.json」というファイルが保存されます
   - iPhoneの場合：「"ファイル"に保存」を選び、iCloud Driveに入れておくと安心
3. これだけでOK。写真も丸ごとこのファイルに入っています

### データを戻すとき・別の端末に移すとき

「**データを読み込む**」→ 保存しておいたバックアップファイルを選ぶだけで、写真ごと復元されます。
パソコンで登録 → 書き出し → スマホで読み込み、という使い方もできます。
